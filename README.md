# vue-wp-demo-blog
Wordpress + Vue demo blog

Use with Wordpress + http://v2.wp-api.org/ plugin

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
